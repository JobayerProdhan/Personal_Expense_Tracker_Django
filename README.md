# 💰 Personal Expense Tracker

A beginner-friendly **Django** web application for tracking daily expenses. This project helps learners understand Django fundamentals by building a real-world CRUD application.

## ✨ Features

* User Registration & Login
* Add, Edit, Delete Expenses
* Expense Categories
* Search & Filter
* Dashboard with Expense Summary
* Responsive UI
* Django Admin Panel

## 🛠️ Tech Stack

* Python
* Django
* SQLite
* HTML
* CSS
* Bootstrap

## 🎯 Learning Objectives

* Django Models & ORM
* CRUD Operations
* Authentication
* Forms & ModelForms
* Templates & Template Inheritance
* URL Routing
* Static Files
* Django Admin

## 🚀 Run the Project

```bash
git clone https://github.com/your-username/expense-tracker.git

cd expense-tracker

python -m venv venv

venv\Scripts\activate

pip install -r requirements.txt

python manage.py migrate

python manage.py createsuperuser

python manage.py runserver
```

Open: `http://127.0.0.1:8000/`
